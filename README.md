# The Carnival Forge

A creative engine for building worlds, books, games, canon, drafts, tasks, and exports.

## Version

v0.1 — Project Core

## Current Features

- Create projects
- Open project workspaces
- Save notes
- Save canon entries
- Save tasks
- Auto-save to browser localStorage
- Export project backup as JSON
- Import project backup from JSON

## Planned Features

- Draft manager
- AI prompt builder
- PDF export
- Continuity checker
- Game builder modules
- Character sheets
- Lore database
- Timeline system
- GitHub/cloud sync

## First Goal

Create a working browser-based creative command center that can store and organize project information without needing a paid backend.
